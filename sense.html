Sensing

The functions listed on this page are related to the Sensing of different variables on the buggy.

LSpeed
float Rspeed()

This function will record the RPS of the left motor from the encoder and calculate its speed.

RSpeed
float Rspeed()

This function will record the RPS of the right motor from the encoder and calculate its speed.

Position
int position()

This position checks the position of the buggy by taking the inputs from all 6 sensors and comparing their values. It will be called from the forward function and will return an integer to indicate the white line's position relative to the buggy. 0 means the line is in the centre, 1 means the line is the left of the buggy, and 2 means the line is to the right of the buggy.

Slope
int slope()

This function will check whether the buggy is going up a slope, in which case the motor will be switch to bipolar mode for a higher torque. It will achieve this by reading the value from the accelerometer on the Application Shield and return its status as an integer. 0 means on flat ground, 1 means going uphill and 2 means going downhill.

Bluetooth Input
char bluetoothinput()

This function will take the input from the BLE module if there is a new input and return the input as a char, if a specific character is inputted, the buggy will stop immediately, this is for the user to stop the buggy if they see the buggy isn't functioning properly.
