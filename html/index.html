Overview

This is an overview for the software of the Embedded System Project by Group 37 of UoM EEE

Purpose

The purpose of this software is to drive a buggy autonomously along a track with white centre lines.

The buggy will achieve this by taking the reading from the sensors and adjust the PWM signal sent to the Left and Right motors accordingly.

Hardware

The software is written for the STM32F401RE Nucleo Platform with the UM1601 Interface board.

Six TCRT5000 sensors are used.
